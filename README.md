# UTW - Software Development Kit

## Description
This repository contains the **SDK for managing dynamic resources** in Unity, specifically designed for the **UTW tank simulator**. The SDK allows users to easily create, manage, and export **Asset Bundles** through an enhanced Unity Editor workflow. The project serves as a starting point for organizing dynamic resources and optimizing asset management for game development.

## Features
- **Custom Unity Editor Scripts**: A set of custom editor scripts that extend the Unity Editor to simplify the creation of dynamic resources and Asset Bundles.
- **Asset Bundle Creation**: Easily convert prefabs to Asset Bundles with a single click.
- **Project Management**: The Unity project provides a workspace for managing and organizing multiple projects. Users can create or open existing projects directly in the Unity Editor.
- **Flexible Asset Management**: Enables efficient handling of dynamic assets, reducing memory usage and improving performance.

## Prerequisites
To use this SDK, you'll need the following:
- Unity (version X.X.X or higher)
- Basic knowledge of Unity Editor and Asset Bundles
- A compatible operating system (Windows, macOS, or Linux)

## Installation
1. **Download the Unity Project**:
   - Clone or download this repository.
   - Open the Unity project using the Unity Hub or directly in the Unity Editor.

2. **Setting up the Editor**:
   - Open the project in Unity and wait for all dependencies to be imported.
   - Once the project is loaded, you can start creating Asset Bundles using the provided editor tools.

3. **Creating a New Project**:
   - Upon opening the Unity project, you will be prompted with an option to either **create a new project** or **open an existing one**. Select your preferred option to begin working on your project.

4. **Creating Asset Bundles**:
   - Use the provided editor scripts to create prefabs.
   - Click the button to convert the prefabs into Asset Bundles and export them for use in your project.

## Usage
- **Creating a Prefab**: Use the custom editor window to easily create a new prefab with specific settings.
- **Building Asset Bundles**: After creating the prefab, click the button in the editor to generate the corresponding Asset Bundle.
- **Managing Projects**: The project management system allows you to save and load different resource projects, keeping all your work organized in one Unity project.
